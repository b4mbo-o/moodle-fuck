# Moodle-fucker

`Moodle-fucker` は、Moodle上のクイズ問題を検出して `capi.voids.top/v2` に送り、回答候補をページ上に表示する Chrome 拡張です。

## できること

- 現在のページから問題文と表示中の選択肢を検出する
- 検出した内容を AI API に送って回答候補を生成する
- 返ってきた回答を画面右下のポップアップに表示する
- API キーなしでそのまま使える

## ファイル構成

```text
moodle-fuck/
|-- manifest.json
|-- content.js
|-- background.js
|-- README.md
|-- icon.png
`-- data/
    `-- main.py
```

## セットアップ

1. Chrome で `chrome://extensions/` を開く
2. デベロッパーモードを有効にする
3. `パッケージ化されていない拡張機能を読み込む` を押す
4. このフォルダを選択する

## 使い方

1. Moodle のクイズページを開く
2. ページの読み込みが終わるまで待つ
3. 拡張機能が問題文と選択肢を読み取る
4. `background.js` から `https://capi.voids.top/v2` に問い合わせる
5. 回答候補がページ右下のポップアップに表示される

## メモ

- 現在の実装では サードパーティ製のAIを利用しています。そのためAPIKeyは不要です。
- デフォルトモデルは `gpt-4o-2024-11-20` を使っています
- 1つ目のエンドポイントで失敗した場合は、互換エンドポイントへ順にフォールバックします
